# Description
SoLong is a project where you need to make a simple
2D game using the MacroLibX.
it aims to improve your skills in window management,
event handling, colors, textures, etc.

# Instructions
Clone the MacroLibX (https://github.com/seekrs/MacroLibX.git) into 
the root of the project then run make in the root of the project.
